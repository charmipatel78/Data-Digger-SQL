# 📊 DataDigger - SQL Project

## 📌 Overview

DataDigger is a MySQL-based database project designed to manage and analyze business data including customers, orders, products, and order details. It demonstrates database design, data manipulation, and analytical querying.

---

## 🗂️ Database Structure

The project includes the following tables:

* **Customers** – Stores customer information
* **Orders** – Contains order transactions
* **Products** – Holds product details and pricing
* **OrderDetails** – Tracks product-wise order data

Database creation and structure are defined in:

* `datadigger project database.sql` 

---

## ⚙️ Features

* Database creation and table relationships
* Insert, Update, Delete operations (CRUD)
* Data filtering and sorting
* Aggregate functions (MAX, MIN, AVG, SUM)
* Business insights queries

---

## 📈 Sample Analysis Performed

* Highest, Lowest, and Average Order Amount
* Total Revenue Calculation
* Top Selling Products
* Recent Orders Analysis
* Product Price Comparison

(Executed queries available in project file) 

---

## 🛠️ Technologies Used

* MySQL
* SQL (DDL, DML, Aggregate Functions)

---

## 🚀 How to Run

1. Open MySQL Workbench or Command Line
2. Run the database script:

   ```sql
   source datadigger project database.sql;
   ```
3. Execute queries from the project file

## 📌 Conclusion

This project demonstrates practical SQL skills including database design, data manipulation, and extracting meaningful insights from structured data.
